InstallationÂ¶

cellDancer requires Python version >= 3.7.6 to run.

To run the notebook locally, create an conda or Anaconda environment as conda create -n cellDancer python==3.7.6, and activate the new environment with conda activate cellDancer. cellDancer could be installed with pip install celldancer.

To install cellDancer from source code, run 'pip install your_path/cellDancer'.

The dependencies could also be installed by pip install -r requirements.txt.

For M1 Mac users if you encountered a problem while installing bezier. Please refer to the following link: https://docs.conda.io/en/latest/.
